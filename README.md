## Description

This is simple currency exchange web calculator that converts PLN to other currencies such as EUR, USD, CHF or GBP.
The application was created with the use of my current knowledge :)

### Technologies used
-	HTML
-	CSS
-	BEM
-	JavaScript
- Normalize
-	ES6+ features
